Print a Drawing of Your Design

To print a black-and-white drawing of your current design:

  1. Click the File menu, and then select the .
  2. On the Print dialog box, select the printer you want to use, enter the number of copies you want to print, and choose the portrait or landscape format. (The portrait format is preferable for most designs.) Then click OK.

To print the drawing using the currently selected printer:

Click the on the Main Toolbar.

Notes and Tips

The printed picture shows the configuration of your truss, annotated with and dimensions. Member properties are shown in tabular format at the bottom of the page.

The first time the Print button is pressed, the Printer Selection Dialog will be shown so that you can select the desired printer. Later printouts will be sent directly to the selected printer with no dialog.
