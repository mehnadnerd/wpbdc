Open an Existing Bridge Design File

To open an existing bridge design file:

  1. Click the on the Main Toolbar.
  2. Choose the disk drive, folder, and filename you want to open, then click OK.

Notes and Tips

The default file extension for bridge design files created in the Bridge Designer 2016 is .bdc.

The Bridge Designer 2016 cannot read the .bdf , .bd4, and bdc bridge design files created with earlier versions of the software. Sorry. Backward compatibility is impossible with BD, because of its use in an annual bridge design contest.

If your current design has not been saved, you will be prompted to save it before you open a file.
