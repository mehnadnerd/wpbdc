Go To Iteration button

 Click the Go To Iteration button to display the Design Iteration Browser. Use the Browser to display on the Drawing Board any design iteration created during the current . Once the Browser is displayed, select the desired iteration, and click OK.

Use the Tree View tab to see how design iterations are related.

Notes and Tips

The Go To Iteration button is located on the Main Toolbar. It can also be accessed from the Edit menu.

The Design Iteration Browser provides a Preview window to help you select the correct iteration before actually displaying it on the Drawing Board.

Previous design iterations are only preserved until the end of the current session. When you start a new design or exit the Bridge Designer 2016, previous iterations are deleted. Thus the Design Iteration Browser can only display iterations created during the current session.
