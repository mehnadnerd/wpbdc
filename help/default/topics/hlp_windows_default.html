Windows Default Printer

To set the Windows default printer, click the Start button on the Windows task bar. The click Settings, then Printers. In the Printers dialog box, highlight the name of the printer that you wish to set as the default. Now click the File menu. If the printer you selected is already the default, there will be a check mark next to the Set As Default item in the File menu. If not, click Set As Default, and your selected printer will become the Windows default.
