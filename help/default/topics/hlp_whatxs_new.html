What's New in the Bridge Designer 2016

The Bridge Designer 2016 is a special edition of the BD software, developed for the 2016 Bridge Design Contest (http://bridgecontest.org). This is the only version of BD that can be used to enter the 2016 contest.

The user interface and functions of the Bridge Designer 2016 are essentially identical to those of the Bridge Designer 2013, which preceded it. The only significant changes in BD 2016 are as follows:

  • In BD 2016, one of the two truck loading configurations has changed. The two-lane loading is still a pair of AASHTO H25 trucks weighing 225 kN each; however, the single-lane permit loading now weighs 480 kN. The AASHTO truck loading has a heavy axle and a lighter axle, while the permit loading uses equal axle loads. Both loadings use a dynamic load allowance of 1.33. As a result of this change, it is not clear which of the two load configurations is more severe.

  • BD 2016 uses new cost factors to calculate the cost of your design. Thus the costs of bridges designed with BD 2016 have no comparability with the costs of bridges designed with earlier versions.

  • BD 2016 allows for both four-character and six-character Local Contest Codes, to be used in conjunction with the Bridge Design Contest.
