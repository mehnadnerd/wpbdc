Save the Current Design

At any time during the design process, you can save your current design as a . There are three different ways to save a design.

If you have not previously saved the design:

  1. Click the on the Main Toolbar.
  2. Choose the disk drive and folder in which you want to save the file.
  3. Enter a file name, then click OK.

If you have previously saved the design, and you do not want to change the file name:

Click the on the Main Toolbar. Your existing bridge design file will be overwritten with your new design, and the file name will remain the same.

If you have previously saved the design, but you want to change its file name:

  1. Click the in the File menu.
  2. Choose the disk drive and folder in which you want to save the file.
  3. Enter a new file name, then click OK.

Notes and Tips

The default file extension for bridge design files created in the Bridge Designer 2016 is .bdc. If you do not specify a file extension when you enter the file name, the .bdc extension will be added automatically.

We strongly recommend that you use the default .bdc extension for all bridge design files.

Bridge design files created in Bridge Designer 2016 cannot be read by earlier versions of the software.
