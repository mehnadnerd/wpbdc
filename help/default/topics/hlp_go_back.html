Go Back button

 Click the Go Back button to display the previous design iteration on the Drawing Board.

Notes and Tips

The Go Back button is located on the Main Toolbar. It can also be accessed from the Edit menu.

The Go Back button can only be used to display design iterations created during the current .

Once you have used the Go Back button to display a previous design iteration, you can use the to display a more recent iteration.

You can also use the to display any design iteration created during the current session
