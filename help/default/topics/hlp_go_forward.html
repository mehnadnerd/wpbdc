Go Forward button

 Click the Go Forward button to display a more recent design iteration on the Drawing Board.

Notes and Tips

The Go Forward button is located on the Main Toolbar. It can also be accessed from the Edit menu.

The Go Forward button can only be used to display design iterations created during the current .

The Go Forward button is only activated after you have used the at least once.

You can also use the to display any design iteration created during the current session.
