Decrease Member Size button

 Click the Decrease Member Size button to decrease the of the currently selected member(s) to the next smaller size.

Notes and Tips

The Decrease Member Size button is located on the Member Properties Toolbar.

If more than one member is selected, clicking this button will decrease the size of all selected , even if they are different sizes. For example, if you have selected a 50mm member, a 90mm member, and a 120mm member simultaneously, clicking the Decrease Member Size button will change these members to 45mm, 80mm, and 110mm, respectively.

When you use the Decrease Member Size button, the is updated to reflect the change.

When you use the Decrease Member Size button, the other two (material and cross-section type) do not change.
